# BANKAPI  

API developed in Node with GraphQL and Apollo integration for a banking dashboard with real-time transactions.  


---  
## PROJECT SETTINGS ⚙️    

➜ npm install  
➜ npm run dev  

You need to create a .env file for configuration:
```javascript
DB_USER
DB_PASS
DB_NAME
DB_HOST
DB_PORT
PORT
```
